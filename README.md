## Authentication System 

This is an Authentication System with a full stack aplication

  - backend: 
      - nodeJS;
      - prisma;
      - JWT;
      - typescript;
      - middlewares;
      
      Ps: to run the server: "npm run dev"
    
   - frontend:
      - reactJS;
      - typescript;
      - context api;
      - axios;
      - localStorage;
      
      Ps: to run the server: "npm run dev"
